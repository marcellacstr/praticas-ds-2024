FACTORY PATTERN

Para realizar uma factory pattern é necessário criar uma factory function (factory function é qualquer função que não é uma classe nem um construtor que retorna um novo objeto).

Quando uma função retorna um novo objeto sem a necessidade de utilizar a palavra reservada "new", ela é uma factory function.

História

Necessidade de criar uma classe carro que quando for invocada deve retornar um novo objeto que representa um carro.

Ao utilizar a Factory Pattern, elimina a necessidade da criação de uma classe com construtor em que em sua invocação utiliza a palavra reservada "new" e passa a utilizar uma factory function que cria o mesmo objeto de uma forma mais direta e objetiva.
